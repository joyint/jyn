# Jot — Personal Todo CLI

A fast, Git-native personal todo tool sharing `joy-core` as its foundation. Jot extends the base Item with recurring tasks, due dates, and reminders. Mobile access works through CalDAV (Apple Reminders, Google Calendar) without requiring a native app.
