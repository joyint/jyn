# jyn

[Jyn](https://joyint.com/en/jyn) - a Git-native personal task manager.

Thin alias crate that installs the `jyn` binary directly. All logic lives in
[`jyn-cli`](https://crates.io/crates/jyn-cli); this crate exists so users can
run `cargo install jyn` without the `-cli` suffix.

## Install

```sh
cargo install jyn
```
